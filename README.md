# FakeNewsDetection
A machine learning and NLP-based web application that detects and classifies news articles as real or fake to combat misinformation.

Features
Detects whether a news article is real or fake.
Text preprocessing using NLP techniques such as tokenization, stop-word removal, and stemming.
Machine learning-based classification for accurate predictions.
User-friendly web interface for easy interaction.
Real-time prediction on user-provided news articles.
Dataset

The dataset used for training and evaluation was obtained from Kaggle. Due to the large size of the dataset files, they have not been included in this repository. You can download the dataset directly from Kaggle and place it in the appropriate project directory before running the application.

Note

Only the source code and necessary project files are included in this repository to keep it lightweight and maintainable.
